Research Projects

Forecasting soil moisture values across Nepal using satellite images via remote sensing

We accessed the datasets provided by NASA SMAP L4 3-hourly 9-km surface and root zone soil moisture using google earth engine and augmented it into an image per day. Then, we created a CNN-LSTM model to analyze the past trends and forecast future moisture values. As the input was intensive, we used several activation functions and optimization methods to make it run using google colab. After months of model evaluation, model training and changes, expected predictions were met. The manuscript is almost ready and once its done, it will be sent to a peer reviewed journal for review/publication.
